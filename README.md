# README

## clone project

```
git clone https://github.com/sbrcode/z-rn-project.git
cd z-rn-project
```

## initialize project with dependencies

```
yarn install
```

## start project

```
yarn start
```

## Download "Expo Go" into your device

- Android: open Expo Go and scan QR Code
- iOS: open camera and scan QR Code
